# Elyth

Laravel inspired HTTP Server starter template, powered by Bun & Elysia.

## Features

1. MVC Architecture
2. Router & middlewares
3. SSR with TSX + Vite
4. Modular & extensible

## Getting Started

1. Download the f& extract it
2. Run `bun install`
3. Run the development server by `bun mvc serve`
